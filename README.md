# Intuition Test

A multi-page Python desktop app built with CustomTkinter.
Test your sixth sense across 10 mystery image questions!

## Features
- User profile with dynamic expected score algorithm
- 10 randomized mystery image questions with reveals
- Real-time scoring and result comparison
- Clean dark UI with color-coded ranks

## How to Run
pip install customtkinter pillow
python intuition_game.py

## Built With
Python, CustomTkinter, Pillow
