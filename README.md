# 🧠 Intuition Game

An interactive Python-based intuition game built with CustomTkinter. 
Test your instincts across 10 mystery scenarios — each one reveals 
something unexpected based on your choices. Receive a final score 
with personalized feedback at the end.

Built as a semester 1 project at Jinnah University for Women.

---

## 🎮 What It Does

- Presents 10 visual mystery scenarios (cave, fog, sea, sky, and more)
- Player picks their instinctive answer
- Each scenario reveals a surprise image based on the response
- Final score calculated with custom feedback message

---

## 🖥️ How To Run

**Step 1 — Install dependency:**
```bash
pip install customtkinter
```

**Step 2 — Run the game:**
```bash
python intuition_game.py
```

---

## 🛠️ Built With

- Python
- CustomTkinter (GUI)
- PIL / Tkinter (image handling)
